
The Linguistic C++ library is made up of classes
that represent linguistic concepts and have these
attributes:

    * independence from the relational database
	* independence from the gerdem, sql, odbcsql,
	  orasql and stemgen libraries.
	  
This library is dependent upon the following:

    * utility library
	* ltx library
	* tm library
	* entity library
	* pattern matcher library
	* finish libraries
	* rule engine library
